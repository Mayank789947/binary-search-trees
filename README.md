# binary-search-trees

A project on binary search tree algorithm with following methods :-

To check all the methods of this project run the following command in your terminal 
# command 
- node BinarySearchTree.js


# methods
buildTree method - build a balanced binary tree from an array

includes method - Check whether a values present or not inside a tree

insert method - Insert new value inside a tree and keep the balance of the tree after that as well

deleteItem method - delete a node and balanced the remaining nodes so that tree will always be balanced

levelOrderForEach method - Traverse the tree in breadth-first and return the tree values in level order.

inOrderForEach method - Traverse the tree and  give you value in sorted manner always.

preOrderForEach method - Traverse the tree in depth-first order and give values of a tree.

postOrderForEach method - Similar to preOrder, It also traverse the tree in depth-first order and return the tree values.

height method - Return the height of a balanced binary tree.

depth method - Return the depth of a balanced binary tree.

isBalanced method - Check whether the tree is balanced or not.

rebalance method - When calling this method it rebalanced the unbalanced tree
